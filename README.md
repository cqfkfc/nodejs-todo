# nodejs-todo
 
 ## To install
 `npm install`
 
 ## To start
 `node todo-finder.js` (defaults to test-example directory)
 

 `node todo-finder.js .` (runs the application at the current directory)
 
 `node todo-finder.js "C:\xxx\xx\xx"` (runs the application at the defined directory)
 
 ## To run tests
 `npm test`
